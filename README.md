# 英雄联盟语音

英雄语音集合

## 项目概述

提供了一个直观、美观的界面，可以浏览英雄联盟中的各个英雄，并聆听他们的语音台词。

## 项目结构

```
voice/
├── index.html         # 主页面
├── style.css          # 样式文件
├── script.js          # 主要JavaScript逻辑
└── static/            # 静态资源目录
    ├── background.jpg # 背景图片
    ├── banner.jpg     # 顶部横幅图片
    ├── icon/          # 英雄头像和图片资源
    └── voice/         # 各英雄语音文件
```
